# 循环神经网络 (RNN)

## 问题描述

利用循环神经网络, 实现唐诗生成任务.

## 数据集

唐诗 (`tangshi.txt`).

## 题目要求

tensorflow: 补全程序, 主要是前面的 3 个空和生成诗歌的一段代码.

pytorch: 补全对应的 `rnn.py` 文件中的两处代码.

生成诗歌, 开头词汇是 "日、红、山、夜、湖、海、月".
